# STT-Cameroun Digital Lead Engine 🚀

This repository contains the digital architecture, landing page code, and analytics configurations for the STT-Cameroun Artificial Intelligence & Data (DQP) student acquisition funnel. 

Developed and maintained by **AbbaTek Group**.

## 🎯 Project Objective
To intercept high-intent Google search traffic for AI and IT diplomas in Yaoundé and convert them into qualified WhatsApp inquiries for the STT admissions team using a performance-based Cost-Per-Lead (CPL) model.

## 🏗️ Architecture Overview
* **Hosting / Domain:** `promo.stt-cameroun.com` (Subdomain Strategy)
* **Frontend:** [HTML5, Tailwind CSS]
* **Analytics Engine:** Google Tag Manager (GTM) + Google Analytics 4 (GA4)
* **Conversion Mechanism:** WhatsApp Business API Direct Routing

## 📈 The Funnel Flow
1. **Traffic Source:** Local SEO & Google Business Profile.
2. **Landing Page:** High-conversion, mobile-optimized capture page.
3. **Action:** User clicks the primary Call-to-Action (CTA).
4. **Tracking:** GTM fires a conversion event (AbbaTek compensation trigger).
5. **Handoff:** User is redirected to STT's WhatsApp with a pre-filled intention message.

# Moteur d'Acquisition Numérique STT-Cameroun 🚀

Ce dépôt contient l'architecture numérique, le code de la page de destination (landing page) et les configurations d'analyse pour le tunnel d'acquisition d'étudiants en Intelligence Artificielle & Data (DQP) de STT-Cameroun. 

Développé et maintenu par **AbbaTek Group**.

## 🎯 Objectif du Projet
Intercepter le trafic de recherche Google à forte intention pour les diplômes en IA et Informatique à Yaoundé et le convertir en prospects WhatsApp qualifiés pour l'équipe d'admission de STT, en utilisant un modèle au Coût Par Lead (CPL) basé sur la performance.

## 🏗️ Aperçu de l'Architecture
* **Hébergement / Domaine :** `promo.stt-cameroun.com` (Stratégie de sous-domaine)
* **Frontend :** HTML5, Tailwind CSS, et JavaScript (Vanilla)
* **Moteur d'Analyse :** Google Tag Manager (GTM) + Google Analytics 4 (GA4)
* **Mécanisme de Conversion :** Routage Direct WhatsApp Business API

## 📈 Déroulement du Tunnel (Funnel Flow)
1. **Source de Trafic :** SEO Local & Fiche Google My Business.
2. **Page de Destination :** Page de capture à fort taux de conversion, optimisée pour la navigation mobile (3G/4G).
3. **Action :** L'utilisateur clique sur le bouton principal d'Appel à l'Action (CTA).
4. **Suivi (Tracking) :** GTM déclenche un événement de conversion (Déclencheur de la rémunération AbbaTek).
5. **Transfert Automatique :** L'utilisateur est redirigé vers le WhatsApp officiel de STT avec un message d'intention pré-rempli.
